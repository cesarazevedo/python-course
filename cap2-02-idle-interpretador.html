💻 Usando o Interpretador (IDLE)

Seção 2.2 — Seu playground para testar código Python

🎮 O que é o IDLE?

O IDLE é a interface gráfica que acompanha o Python. Ele permite editar e executar programas de forma simples, tudo num só lugar.

Como abrir o IDLE:

  • Windows: Menu Iniciar → Python 3.x → IDLE
  • Linux: No terminal, digite idle-python3 ou idle3
  • Mac: No terminal, digite IDLE3 ou idle3

💬 A janela do interpretador

Ao abrir o IDLE, você verá a janela do interpretador (Python Shell). Repare no símbolo >>> — ele indica que o Python está pronto para receber um comando.

Digite seu primeiro comando e pressione Enter:

IDLE - Python Shell >>> print("Oi") Oi >>>

Pronto! Você acabou de executar seu primeiro comando Python. A função print() exibe uma mensagem na tela.

“Uma das grandes vantagens do Python é contar com o interpretador interativo. Você pode testar qualquer comando e ver o resultado instantaneamente.”

🧪 Experimente!

O interpretador é perfeito para experimentar. Tente digitar outros comandos:

IDLE >>> print("Olá, Mundo!") Olá, Mundo! >>> print("Meu nome é Python") Meu nome é Python >>> 2 + 3 5

Perceba que o interpretador também funciona como uma calculadora. Digitou 2 + 3 e ele respondeu 5. Veremos mais sobre isso nas próximas aulas.

Esse ciclo de digitar → executar → ver resultado é muito importante para o aprendizado. Use e abuse do interpretador para testar ideias!
