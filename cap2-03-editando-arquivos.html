Seções 2.3 e 2.4 — Do interpretador ao seu primeiro arquivo .py

📄 Interpretador vs. Editor de Textos

Até agora, digitamos comandos diretamente no interpretador. Isso é ótimo para testes rápidos, mas para programas maiores precisamos de arquivos.

Um programa Python nada mais é que um arquivo-texto escrito em um formato especial (a linguagem Python). Podemos usar qualquer editor de textos: o próprio IDLE, VS Code, Sublime Text, Vim, etc.

⚠ Nunca use o Microsoft Word ou LibreOffice Writer! Esses editores gravam arquivos em formatos especiais que não servem para programar. Use sempre um editor de código ou o próprio IDLE.

💾 Criando seu primeiro arquivo no IDLE

O IDLE tem seu próprio editor de textos embutido. Siga os passos:

  1. Com o IDLE aberto, clique no menu File → New File (Ctrl+N)
  2. Uma nova janela se abre — essa é a janela do editor de textos
  3. Digite seu programa:
    teste.py print("Oi")
  4. Salve com File → Save (Ctrl+S). Nomeie como teste.py
  5. Execute com Run → Run Module (F5)

O resultado aparecerá na janela do interpretador (Python Shell). Você verá a palavra RESTART seguida do resultado Oi.

💡 Lembre-se: a extensão .py não é adicionada automaticamente pelo IDLE. Sempre salve seus programas com a extensão .py!

⚠ Cuidados ao digitar programas

Ao digitar um programa, preste atenção nestes detalhes:

  • Maiúsculas e minúsculas são diferentes — print funciona, Print dá erro. Python é case-sensitive
  • Aspas devem ser abertas e fechadas — toda vez que abrir aspas, feche-as. O IDLE muda a cor do texto entre aspas para ajudar
  • Parênteses não são opcionais — não remova os parênteses das funções. Preste a mesma atenção que dá às aspas
  • Espaços são muito importantes — Python usa a indentação (espaços no início da linha) para organizar o código. Nunca junte duas linhas em uma só

“Organize seus programas em uma pasta específica no computador. Crie um diretório para cada capítulo. Com o tempo, essa organização vai facilitar muito.”
