🧰 O que é o interpretador?

Antes de programar, precisamos instalar o interpretador Python. Ele é o programa que aceita comandos escritos em Python e os executa, linha a linha. É ele quem traduz nossos programas em um formato que o computador entende.

Sem o interpretador, nossos programas seriam apenas texto. Ele também verifica se escrevemos corretamente, mostrando mensagens de erro quando encontra algum problema.

“O interpretador Python é como um tradutor: você fala em Python, ele traduz para o computador executar.”

🖥 Instalando no Windows

O Python está disponível gratuitamente no site oficial. Veja o passo a passo:

  1. Acesse python.org e clique em Downloads
  2. Baixe a versão mais recente do Python 3 (ex: Python 3.12)
  3. Execute o instalador baixado (.exe)
  4. IMPORTANTE: Marque a opção “Add Python to PATH” antes de clicar em Install
  5. Escolha “Install for all users” (recomendado)
  6. Aceite o diretório padrão ou escolha outro local
  7. Aguarde a barra de progresso completar
  8. Clique em Finish para concluir

⚠ Dica importante: Marcar “Add Python to PATH” evita muita dor de cabeça depois. Se esqueceu, reinstale marcando essa opção.

🐧 Instalando no Linux

A maioria das distribuições Linux já vem com Python instalado. Para verificar a versão, abra o terminal e digite:

Terminal python3 --version

Se precisar instalar ou atualizar no Ubuntu/Debian:

Terminal sudo apt-get install python3 sudo apt-get install idle-python3

Você precisa de permissões de administrador (root) para instalar via apt-get.

🍎 Instalando no Mac OS X

Os Macs vêm com uma versão do Python, mas geralmente é a versão 2.x (antiga). Para instalar o Python 3, você pode:

  • Baixar o instalador diretamente do python.org (formato .pkg)
  • Usar o Homebrew: brew install python3
  • Usar o MacPorts: sudo port install python34

Para executar o interpretador após instalar, digite python3 no terminal.
